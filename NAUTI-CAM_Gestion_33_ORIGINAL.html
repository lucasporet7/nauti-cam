Configuration email — Brevo (envoi avec pieces jointes)

Configuration Brevo (gratuit jusqu'a 300 emails/jour) :
1. Creez un compte sur brevo.com (ex-Sendinblue)
2. Allez dans SMTP & API → API Keys
3. Cliquez "Generate a new API key" et copiez la cle
4. Verifiez votre domaine expediteur (ou utilisez votre email directement)

Avantage Brevo : Les PDFs sont envoyes en vraie piece jointe dans l'email.
Comment heberger la page de signature :
1. Creez un compte gratuit sur github.com
2. Creez un repository public nomme nauti-cam
3. Uploadez le fichier NAUTI-CAM_Signature.html
4. Allez dans Settings → Pages → Branch: main → Save
5. Copiez l'URL fournie (ex: https://votrecompte.github.io/nauti-cam/NAUTI-CAM_Signature.html)
6. Collez-la dans le champ ci-dessus
